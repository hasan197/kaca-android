# Kaca — Android App

Kirim layar Android ke Mac via WiFi. Capture screen via MediaProjection, kirim JPEG frame ke Mac receiver.

## Build

```bash
./build-android.sh
```

Output: `dist/kaca-android.apk`.

## Cara pakai

1. Jalankan Kaca macOS receiver — QR code muncul di window
2. Buka app Kaca di HP, tap **Scan QR**
3. Setelah IP:PORT terisi otomatis, tap **Start Mirror**
4. Allow screen capture permission
